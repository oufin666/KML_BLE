## 十、注意事项

1. **SD卡引脚**：SDIO使用4位数据总线模式，CMD和D0引脚需要配置上拉电阻
2. **蓝牙串口**：USART1用于BLE通信，波特率通常为115200
3. **GPS串口**：USART2用于GPS模块通信，波特率通常为9600
4. **LCD背光**：PA8和PB1都可以控制LCD背光，根据实际硬件选择
5. **电机PWM**：PC13用于驱动电机，需要配置PWM输出
6. **复用引脚**：PA8和PA10同时用于LCD和SDIO，需要注意使用时的冲突
