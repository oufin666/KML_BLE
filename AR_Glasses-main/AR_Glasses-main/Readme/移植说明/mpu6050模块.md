# MPU6050模块移植说明

## 1. 功能概述

MPU6050模块实现了惯性测量单元(IMU)的功能，包括加速度计和陀螺仪数据的读取、姿态角计算等。主要功能包括：

- MPU6050初始化和自检
- 原始加速度计和陀螺仪数据读取
- 姿态角（Roll、Pitch、Yaw）计算
- 支持硬件I2C和软件I2C两种模式
- 提供多种封装函数，简化使用

## 3. 移植步骤

### 3.1 复制文件

1. 将 `mpu6050.h` 和 `mpu6050.c` 复制到目标项目的 `Drivers/BSP/inc/` 和 `Drivers/BSP/src/` 目录
2. 将 `imu_filter.h` 和 `imu_filter.c` 复制到目标项目的 `Drivers/BSP/inc/` 和 `Drivers/BSP/src/` 目录
3. 将 `mahony_ahrs.h` 和 `mahony_ahrs.c` 复制到目标项目的 `Drivers/BSP/inc/` 和 `Drivers/BSP/src/` 目录

### 3.2 配置I2C模式

在 `mpu6050.h` 文件中，根据硬件连接选择I2C模式：

```c
// I2C模式选择
#define MPU6050_USE_SOFT_I2C    1   // 1: 使用软件I2C, 0: 使用硬件I2C
```
- 硬件I2C模式：需要在CubeMX中配置I2C1接口
- 软件I2C模式：需要在CubeMX中配置GPIO引脚（默认PB8->SDA, PB9->SCL）

### 3.3 启用IMU滤波器

在 `main.h` 文件中，添加以下宏定义：

```c
// IMU滤波器配置
#define ENABLE_IMU_FILTER 1  // 1: 启用IMU滤波器, 0: 禁用IMU滤波器
```
//作者说：这里建议不要开启滤波，因为对于我们识别翻腕来讲，不需要滤波已经很准确了，加个滤波反而变复杂，没必要

### 3.4 修改 freertos.c

1. 添加必要的包含：

```c
#include "mpu6050.h"
#include "imu_filter.h"
#include "mahony_ahrs.h"
```

2. 在任务中使用MPU6050相关函数（例如在StartTaskHR任务中）：

**使用示例：**

```c
// 任务函数示例1：完整数据读取
void StartTaskHR(void *argument)
{
    MPU6050_RawData_t raw_data;
    uint8_t mpu_initialized = 0;
    uint32_t last_mpu_tick = 0;
    
    // 1. 初始化MPU6050（只需调用一次）
    #if MPU6050_USE_SOFT_I2C
    if (MPU6050_SystemInit(NULL, BLE_UART_HANDLE, &raw_data) == 0) {
    #else
    if (MPU6050_SystemInit(&hi2c1, BLE_UART_HANDLE, &raw_data) == 0) {
    #endif
        mpu_initialized = 1;
    } else {
        char tx_buffer[128];
        sprintf(tx_buffer, "[System] Initialization failed!\r\n");
        HAL_UART_Transmit(BLE_UART_HANDLE, (uint8_t*)tx_buffer, strlen(tx_buffer), 100);
    }
    
    // 2. 主循环
    while (1) {
        // 每500ms读取一次数据
        if (mpu_initialized && (HAL_GetTick() - last_mpu_tick > 500)) {
            last_mpu_tick = HAL_GetTick();
            
            // 读取数据并计算姿态角
            #if MPU6050_USE_SOFT_I2C
            MPU6050_ReadAndCalculateAttitude(NULL, BLE_UART_HANDLE, &raw_data, NULL, NULL, NULL);
            #else
            MPU6050_ReadAndCalculateAttitude(&hi2c1, BLE_UART_HANDLE, &raw_data, NULL, NULL, NULL);
            #endif
        }
        
        osDelay(10); // 任务延时
    }
}

// 任务函数示例2：仅检测翻腕（使用封装函数）
void StartWristRaiseTask(void *argument)
{
    // 直接调用封装的翻腕检测任务函数
    // 参数1：I2C句柄（硬件I2C模式使用，软件I2C模式传NULL）
    // 参数2：串口句柄，用于输出调试信息
    // 参数3：检测间隔（毫秒）
    #if MPU6050_USE_SOFT_I2C
    MPU6050_WristRaiseTask(NULL, BLE_UART_HANDLE, 100);
    #else
    MPU6050_WristRaiseTask(&hi2c1, BLE_UART_HANDLE, 100);
    #endif
}
```

## 4. 函数说明

### 4.1 核心函数

| 函数名 | 说明 | 使用场景 |
|-------|------|----------|
| `MPU6050_RunSelfTest` | 运行自检，初始化并测试数据读取 | 系统启动时验证MPU6050是否正常工作 |
| `MPU6050_StartDataMonitorTask` | 启动数据监控任务，循环读取数据并打印 | 在独立任务中持续监控MPU6050数据 |
| `MPU6050_InitAndRead` | 初始化和数据读取封装函数 | 需要一次性读取MPU6050数据时使用 |
| `MPU6050_ReadAndCalculateAttitude` | 读取数据并计算姿态角 | 需要获取姿态角时使用 |
| `MPU6050_SystemInit` | 系统初始化函数，包含I2C、IMU滤波器和MPU6050的初始化 | 系统启动时进行MPU6050相关初始化 |
| `MPU6050_GetRoll` | 获取Roll角（横滚角） | 只需要获取Roll角时使用 |
| `MPU6050_GetPitch` | 获取Pitch角（俯仰角） | 只需要获取Pitch角时使用 |
| `MPU6050_DetectWristRaise` | 检测翻腕状态 | 智能手表等设备需要检测翻腕动作时使用 |
| `MPU6050_WristRaiseTask` | 封装的翻腕检测任务函数 | 在专门的任务中实现完整的翻腕检测功能时使用 |

### 4.2 函数详细说明

1. **MPU6050_RunSelfTest**
   - 功能：执行MPU6050的初始化和数据读取测试
   - 参数：
     - `hi2c`：I2C句柄指针（硬件I2C模式使用，软件I2C模式传NULL）
     - `huart`：串口句柄指针，用于输出调试信息
     - `raw_data`：用于存储读取的原始数据（可为NULL）
   - 返回值：0-成功，1-初始化失败，2-数据读取失败

2. **MPU6050_StartDataMonitorTask**
   - 功能：启动数据监控任务，循环读取MPU6050数据并通过串口打印
   - 参数：
     - `hi2c`：I2C句柄指针（硬件I2C模式使用，软件I2C模式传NULL）
     - `huart`：串口句柄指针，用于输出调试信息
   - 返回值：无

3. **MPU6050_InitAndRead**
   - 功能：初始化MPU6050并读取数据
   - 参数：
     - `hi2c`：I2C句柄指针（硬件I2C模式使用，软件I2C模式传NULL）
     - `huart`：串口句柄指针，用于输出调试信息
     - `raw_data`：存储原始数据的结构体指针
   - 返回值：0-成功，1-失败

4. **MPU6050_ReadAndCalculateAttitude**
   - 功能：读取MPU6050数据并计算姿态角
   - 参数：
     - `hi2c`：I2C句柄指针（硬件I2C模式使用，软件I2C模式传NULL）
     - `huart`：串口句柄指针，用于输出调试信息
     - `raw_data`：存储原始数据的结构体指针
     - `roll`：存储Roll角的指针（可为NULL）
     - `pitch`：存储Pitch角的指针（可为NULL）
     - `yaw`：存储Yaw角的指针（可为NULL）
   - 返回值：0-成功，1-失败

5. **MPU6050_SystemInit**
   - 功能：系统初始化，包含I2C、IMU滤波器和MPU6050的初始化
   - 参数：
     - `hi2c`：I2C句柄指针（硬件I2C模式使用，软件I2C模式传NULL）
     - `huart`：串口句柄指针，用于输出调试信息
     - `raw_data`：存储原始数据的结构体指针
   - 返回值：0-成功，1-失败

6. **MPU6050_GetRoll**
   - 功能：获取Roll角（横滚角）
   - 参数：
     - `hi2c`：I2C句柄指针（硬件I2C模式使用，软件I2C模式传NULL）
     - `huart`：串口句柄指针，用于输出调试信息（可为NULL）
   - 返回值：Roll角（度），返回0表示获取失败
   - 说明：调用此函数会自动读取数据并计算Roll角

7. **MPU6050_GetPitch**
   - 功能：获取Pitch角（俯仰角）
   - 参数：
     - `hi2c`：I2C句柄指针（硬件I2C模式使用，软件I2C模式传NULL）
     - `huart`：串口句柄指针，用于输出调试信息（可为NULL）
   - 返回值：Pitch角（度），返回0表示获取失败
   - 说明：调用此函数会自动读取数据并计算Pitch角

8. **MPU6050_DetectWristRaise**
   - 功能：检测翻腕状态（通过检测pitch角的大变化）
   - 参数：
     - `current_pitch`：当前的pitch角（度）
   - 返回值：1-检测到翻腕，0-未检测到翻腕
   - 说明：基于pitch角的变化检测翻腕动作，适用于智能手表等设备

9. **MPU6050_WristRaiseTask**
   - 功能：封装的翻腕检测任务函数
   - 参数：
     - `hi2c`：I2C句柄指针（硬件I2C模式使用，软件I2C模式传NULL）
     - `huart`：串口句柄指针，用于输出调试信息
     - `interval`：检测间隔（毫秒）
   - 返回值：无
   - 说明：此函数实现了完整的翻腕检测功能，包括初始化、数据读取和翻腕检测，适用于在专门的任务中使用

## 5. 使用方法

### 5.1 核心使用流程

1. **初始化**：调用 `MPU6050_SystemInit` 函数完成所有初始化工作（只需调用一次）
2. **数据读取**：在主循环中使用 `MPU6050_ReadAndCalculateAttitude` 函数读取数据并计算姿态角
3. **功能使用**：根据需要使用相应的功能函数，如：
   - 单独测试：`MPU6050_RunSelfTest`
   - 数据监控：`MPU6050_StartDataMonitorTask`
   - 获取单个姿态角：`MPU6050_GetRoll`、`MPU6050_GetPitch`
   - 检测翻腕：`MPU6050_DetectWristRaise`

### 5.2 主要应用场景

**场景1：完整数据读取**
- 适用于需要全面监控MPU6050数据的场景
- 参考示例：`StartTaskHR` 函数

**场景2：仅检测翻腕**
- 适用于智能手表等只需要检测翻腕动作的场景
- 参考示例：`StartWristRaiseTask` 函数

### 5.3 注意事项

- 初始化只需调用一次 `MPU6050_SystemInit` 函数
- 数据读取频率建议在100-500ms之间，避免过于频繁的I2C通信
- 检测翻腕时，建议每100ms获取一次pitch角并进行检测
- 根据实际硬件连接选择I2C模式（硬件I2C或软件I2C）

## 6. 依赖项

### 6.1 硬件依赖

- STM32F4系列微控制器
- MPU6050传感器
- I2C接口（硬件或软件模拟）

### 6.2 软件依赖

- STM32 HAL库
- FreeRTOS（可选，用于任务管理）
- 蜂鸣器和电机驱动（可选，用于系统初始化）

## 7. 注意事项

1. 确保I2C引脚连接正确：
   - 硬件I2C模式：PB8->SDA, PB9->SCL
   - 软件I2C模式：PB8->SDA, PB9->SCL（支持引脚反接的旧版PCB）

2. 确保IMU滤波器和Mahony AHRS算法的采样频率设置正确（默认100Hz）

3. 如遇I2C通信问题，检查I2C时钟频率和硬件连接

4. 移植时注意保持文件路径和包含关系的一致性

## 8. 测试方法

1. 确保MPU6050传感器正确连接
2. 系统启动时，观察串口输出的初始化信息
3. 验证MPU6050数据读取是否正常
4. 测试姿态角计算是否准确
5. 检查系统在不同环境下的稳定性

## 9. 代码结构说明

### 9.1 核心文件

| 文件 | 说明 | 主要功能 |
|------|------|----------|
| `mpu6050.c` | MPU6050驱动实现 | 传感器初始化、数据读取、封装函数 |
| `imu_filter.c` | IMU滤波器实现 | 数据去偏置、滤波处理 |
| `mahony_ahrs.c` | Mahony AHRS算法实现 | 姿态角计算 |

### 9.2 数据流程

1. 系统初始化：`MPU6050_SystemInit` → `MPU6050_RunSelfTest` → `MPU6050_Init`
2. 数据读取：`MPU6050_ReadAndCalculateAttitude` → `MPU6050_InitAndRead` → `MPU6050_GetRawData`
3. 姿态角计算：`IMU_ProcessRawData` → `Mahony_UpdateIMU` → `Mahony_ComputeAngles`
