# 蜂鸣器和电机模块移植说明

## 1. 功能概述

蜂鸣器和电机模块实现了对蜂鸣器和直流电机的控制功能，主要包括：

- 蜂鸣器的初始化和控制（开/关）
- 电机的初始化、速度设置和控制（开/关）
- 支持PWM控制，实现不同频率和占空比

## 2. 需要移植的文件

### 2.1 新增文件

## 3. 移植步骤

### 3.1 复制文件

1. 将 `beep.h` 和 `beep.c` 复制到目标项目的 `Drivers/BSP/inc/` 和 `Drivers/BSP/src/` 目录
2. 将 `motor.h` 和 `motor.c` 复制到目标项目的 `Drivers/BSP/inc/` 和 `Drivers/BSP/src/` 目录

### 3.2 配置GPIO引脚

在CubeMX中配置以下GPIO引脚：

- **蜂鸣器**：PB7（GPIO输出）
- **电机**：PC13（GPIO输出）

### 3.3 配置TIM3

在CubeMX中配置TIM3：

- 时钟源：内部时钟
- 预分频器：83
- 自动重载值：999
- 触发模式：中断
- 频率：8kHz（用于蜂鸣器PWM）

### 3.4 修改 stm32f4xx_it.c

在 `TIM3_IRQHandler` 函数中添加蜂鸣器更新：

```c
void TIM3_IRQHandler(void)
{
  /* USER CODE BEGIN TIM3_IRQn 0 */
  if (!beep_enabled) {
    HAL_TIM_Base_Stop_IT(&htim3); // 如果蜂鸣器未启用，停止中断
    return;
  }
  /* USER CODE END TIM3_IRQn 0 */
  HAL_TIM_IRQHandler(&htim3);
  /* USER CODE BEGIN TIM3_IRQn 1 */
  BEEP_Update();
  /* USER CODE END TIM3_IRQn 1 */
}
```

### 3.5 修改 freertos.c

在任务的主循环中添加电机更新：

```c
// 在主循环中
while (1) {
    // 其他代码...
    
    // 更新电机PWM
    MOTOR_Update();
    
    osDelay(1); // 任务延时
}
```

## 4. 函数说明

### 4.1 蜂鸣器函数

| 函数名 | 说明 | 参数 | 返回值 |
|-------|------|------|--------|
| `BEEP_Init` | 初始化蜂鸣器 | 无 | 无 |
| `BEEP_On` | 开启蜂鸣器 | 无 | 无 |
| `BEEP_Off` | 关闭蜂鸣器 | 无 | 无 |
| `BEEP_Update` | 更新蜂鸣器PWM（在TIM3中断中调用） | 无 | 无 |

### 4.2 电机函数

| 函数名 | 说明 | 参数 | 返回值 |
|-------|------|------|--------|
| `MOTOR_Init` | 初始化电机 | 无 | 无 |
| `MOTOR_On` | 开启电机（默认速度50%） | 无 | 无 |
| `MOTOR_Off` | 关闭电机 | 无 | 无 |
| `MOTOR_SetSpeed` | 设置电机速度 | speed: 速度（0-100%） | 无 |
| `MOTOR_Update` | 更新电机PWM（在主循环中调用） | 无 | 无 |

## 5. 使用方法

### 5.1 蜂鸣器使用

```c
// 初始化蜂鸣器
BEEP_Init();

// 开启蜂鸣器
BEEP_On();

// 关闭蜂鸣器
BEEP_Off();
```

### 5.2 电机使用

```c
// 初始化电机
MOTOR_Init();

// 开启电机（默认速度50%）
MOTOR_On();

// 设置电机速度为75%
MOTOR_SetSpeed(75);

// 关闭电机
MOTOR_Off();
```

### 5.3 完整示例

```c
// 在任务函数中
void StartTask(void *argument)
{
    // 初始化蜂鸣器和电机
    BEEP_Init();
    MOTOR_Init();
    
    while (1) {
        // 开启蜂鸣器和电机
        BEEP_On();
        MOTOR_SetSpeed(50);
        
        osDelay(1000);
        
        // 关闭蜂鸣器和电机
        BEEP_Off();
        MOTOR_Off();
        
        osDelay(1000);
        
        // 更新电机PWM（必须在主循环中调用）
        MOTOR_Update();
    }
}
```

## 6. 注意事项

1. **蜂鸣器**：
   - 使用TIM3中断产生4kHz PWM
   - 当蜂鸣器关闭时，会自动停止TIM3中断，避免无效中断触发

2. **电机**：
   - 使用软件PWM，频率为100Hz
   - 必须在主循环中调用 `MOTOR_Update()` 函数，否则电机不会工作
   - 速度范围为0-100%，0表示停止

3. **GPIO配置**：
   - 确保PB7和PC13引脚已正确配置为GPIO输出
   - 确保TIM3已正确配置

4. **中断处理**：
   - 确保在TIM3_IRQHandler中添加了BEEP_Update()调用
   - 确保在中断中检查beep_enabled标志，避免无效中断

## 7. 故障排除

- **蜂鸣器不响**：
  - 检查PB7引脚连接
  - 检查TIM3配置
  - 检查BEEP_Update()是否在TIM3中断中调用

- **电机不转**：
  - 检查PC13引脚连接
  - 检查MOTOR_Update()是否在主循环中调用
  - 检查速度设置是否大于0

- **中断频繁触发**：
  - 检查beep_enabled标志是否正确设置
  - 确保BEEP_Off()函数被正确调用
